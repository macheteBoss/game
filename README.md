# Инструкция по развёртыванию

- Развернуть git репозиторий
- Поднять контейнеры:
  
  # docker-compose up --build -d

- Выполнить build файл (создаёт юзера для БД, выполняет миграции):

  # ./docker/build.sh (либо bash docker/build.sh)
